How It Works
COCO-SSD — a pre-trained SSD MobileNet model, equivalent to lightweight YOLO, trained on 80 COCO object classes
TensorFlow.js runs the model entirely in your browser — no Python, no server needed
Bounding boxes — coordinates (x,y,w,h) drawn on canvas, same as OpenCV + YOLO in Python
Object tracking — IDs assigned per class instance and tracked across frames by position proximity
